Position Blog

Static: The default value

This property will not work with the top, left, right, bottom properties

Relative

This property will keep the element in the normal flow of elements, it can be used with top, left, right, bottom properties

Absolute

  • This property will place the element outside of the normal flow of elements, it can be used with top, left, right, bottom properties
  • This will position elements relative to the nearest positioned ancestor(meaning: an element with a position set to anything other than static or the intial containing block if none is found)
